Case Study

Cloud cost optimization for a small-scale web application (demo environment)

Reducing Cloud Costs Without Compromising Performance

⚠️ Initial Setup

  • EC2 t2.medium running 24/7
  • No auto-scaling
  • Unused storage volumes
  • No monitoring or alerts

⚙️ Optimization Steps

  • Right-sized instance (t2.medium → t2.small)
  • Removed unused resources
  • Added CloudWatch monitoring
  • Scheduled instance shutdown

📊 Outcome

  • ~30% cost reduction
  • Improved resource utilization
  • Better cost visibility
